Svar:

  1. Webbläsaren anger sidans bredd samma som skärmens bredd. Initial scale=1 betyder att fönstret inte zoomas.
  2. @media screen and (min-width: 60 em) { body { margin: 0 10%; } } Betyder att om skärmens bredd är 60em eller mer så blir margin på body 0 på top och bottom och 10% på right och left
  3. Mobile first och mobile last. Man börjar eller slutar med den minsta och gör det andra efter.
  4. em i media queries?
    Elemenen på webbsidan blir propotionella med storleken på sidans teckensnitt.
  5. Media queries kan användas i style-tagg, extern stilmall eller @import regeln.
